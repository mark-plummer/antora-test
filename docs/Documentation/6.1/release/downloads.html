JDBC Drivers

ThoughtSpot provides the following JDBC drivers:

  • JDK 1.8 + thoughtspot_jdbc42jar

  • JDK 1.7 + thoughtspot_jdbc41.jar

  • JDK 1.6 + thoughtspot_jdbc4.jar

See link:{{ site.baseurl }}/data-integrate/clients/about-jdbc-driver.html[JDBC Driver Overview] on instructions for installing and configuring JDBC drivers.

ODBC Drivers

  • Windows: ThoughtSpot_windows_odbc_4.2.zip + See link:{{ site.baseurl }}/data-integrate/clients/install-odbc-windows.html[Install the ODBC driver on Windows]

  • Linux: ThoughtSpot_odbc_linux_3.4.tar.gz + See link:{{ site.baseurl }}/data-integrate/clients/install-odbc-linux.html[Install the ODBC driver on Linux]
